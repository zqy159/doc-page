前端布局 ​

自适应布局（Adaptive Design） ​

核心机制 ​

通过媒体查询（Media Queries）检测设备特性（如屏幕宽度），根据预设断点加载完全独立的固定布局版本。 例如：当屏幕宽度 < 768px 时，加载移动端布局；当宽度 ≥ 1200px 时，加载桌面端布局。

简述: 自适应布局通过媒体查询加载预设固定布局版本
自适应布局 = 预设断点 + 固定布局切换

技术细节 ​

  • 预设断点：开发者预先定义设备分类（如手机、平板、桌面）。
  • 独立布局：每个断点对应一个完全独立的CSS文件或样式块，覆盖默认样式。
  • 离散切换：布局切换是突变的（如导航栏从水平排列突然变为垂直折叠）。

响应式布局（Responsive Design） ​

核心机制 ​

通过相对单位（如百分比、vw/vh、em/rem） + 弹性布局技术（Flexbox/Grid） + 媒体查询（Media Queries），实现连续、平滑的布局适配。

简述：根据相对单位,弹性布局,以及媒体查询微调适配各个屏幕。
响应式布局 = 弹性单位 + 布局技术 + 连续适配

技术细节 ​

  • 相对单位：定义元素尺寸和间距（如 width: 80% 或 width: 50vw）。
  • 弹性布局：通过Flexbox/Grid使元素自动调整位置和大小。
  • 媒体查询：用于微调样式（如在小屏幕隐藏侧边栏、调整字体大小），而非完全切换布局。
  • 连续适配：布局变化是渐进的（如导航栏从水平排列逐渐变为垂直折叠）。

圣杯布局 ​

定义:圣杯布局是一种三栏布局，中间栏内容优先，两侧栏固定宽度。

html
<div class="container">
    <div class="center">中间栏</div>
    <div class="left">左侧栏</div>
    <div class="right">右侧栏</div>
</div>
css
.container {
  padding: 0 200px 0 200px; /* 为左右侧栏留出空间 */
}

.center, .left, .right {
  float: left;
  position: relative;
}

.center {
  width: 100%;
  background: #eee;
}

.left {
  width: 200px;
  margin-left: -100%;
  left: -200px;
  background: #ff9999;
}

.right {
  width: 200px;
  margin-left: -200px;
  right: -200px;
  background: #99ff99;
}

双飞翼布局 ​

html
<div class="container">
    <div class="center-wrapper">
        <div class="center">中间栏</div>
    </div>
    <div class="left">左侧栏</div>
    <div class="right">右侧栏</div>
</div>
css
.container {
  overflow: hidden; /* 清除浮动 */
}

.center-wrapper, .left, .right {
  float: left;
}

.center-wrapper {
  width: 100%;
}

.center {
  margin: 0 200px 0 200px; /* 为左右侧栏留出空间 */
  background: #eee;
}

.left {
  width: 200px;
  margin-left: -100%;
  background: #ff9999;
}

.right {
  width: 200px;
  margin-left: -200px;
  background: #99ff99;
}

等高布局 ​

html
<div class="container">
  <div class="left">左侧栏</div>
  <div class="center">中间栏</div>
  <div class="right">右侧栏</div>
</div>
css
.container {
  display: flex;
}

.left, .center, .right {
  flex: 1; /* 平均分配剩余空间 */
  padding: 20px;
}

.left {
  background: #ff9999;
}

.center {
  background: #eee;
  margin: 0 20px; /* 添加间距 */
}

.right {
  background: #99ff99;
}

绝对定位布局 ​

html
<div class="container">
  <div class="left">左侧栏</div>
  <div class="right">右侧栏</div>
  <div class="center">中间栏</div>
</div>
css
.container {
  position: relative;
  padding: 0 200px 0 200px;
}

.left {
  position: absolute;
  left: 0;
  width: 200px;
  background: #ff9999;
}

.right {
  position: absolute;
  right: 0;
  width: 200px;
  background: #99ff99;
}

.center {
  margin: 0 200px;
  background: #eee;
}

表格布局 ​

html
<div class="container">
  <div class="row">
    <div class="left">左侧栏</div>
    <div class="center">中间栏</div>
    <div class="right">右侧栏</div>
  </div>
</div>
css
.container {
  width: 100%;
}

.row {
  display: table;
  width: 100%;
  table-layout: fixed; /* 固定列宽，实现等高 */
}

.left, .center, .right {
  display: table-cell;
  padding: 20px;
}

.left {
  width: 200px;
  background: #ff9999;
}

.center {
  background: #eee;
}

.right {
  width: 200px;
  background: #99ff99;
}

弹性盒布局 ​

html
<div class="container">
  <div class="item item1">Item 1</div>
  <div class="item item2">Item 2</div>
  <div class="item item3">Item 3</div>
</div>
css
.container {
  display: flex;
  gap: 20px; /* 项目间距 */
}

.item {
  padding: 20px;
  background: #eee;
}

.item1 {
  flex: 1; /* 占据剩余空间 */
}

.item2 {
  flex: 2; /* 占据两倍剩余空间 */
}

.item3 {
  width: 200px; /* 固定宽度 */
}

Grid布局 ​

html
<div class="container">
  <div class="item item1">Item 1</div>
  <div class="item item2">Item 2</div>
  <div class="item item3">Item 3</div>
</div>
css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 200px; /* 定义三列，比例1:2:固定200px */
  gap: 20px; /* 网格间距 */
}

.item {
  padding: 20px;
  background: #eee;
}
