关键特性 ​

1. 自动排序 ​

Markdown 会自动纠正数字顺序，即使你手动输入不连续的数字（如 1. 3. 5.），最终也会渲染为 1. 2. 3.。 但建议保持数字正确，避免代码可读性下降。

2. 嵌套列表 ​

通过缩进实现多级列表（支持有序+无序混合）：

plain
1. 第一项
   - 子项1（无序）
   - 子项2
3. 第二项
   1. 子项1（有序）
   3. 子项2

渲染效果:

  1. 第一项
    • 子项 1（无序）
    • 子项 2
  2. 第二项
    1. 子项 1（有序）
    2. 子项 2

3. 段落与换行 ​

  • 列表项内换行：在行末加 （两个空格）或空行。
  • 插入非列表内容（如代码块）：需空行分隔。
plain
1. 第一项
   第二行（通过两个空格换行）

2. 第二项
   ```javascript
   console.log("代码块需空行分隔");

渲染结果：

  1. 第一项
    第二行（通过两个空格换行）

  2. 第二项

    javascript
    console.log("代码块需空行分隔");

4. 文字样式 ​

  • 用 双星号 或 双下划线 包裹文字，渲染后文字会加粗
  • 用 单星号 或 单下划线 包裹文字，渲染后文字会倾斜
  • 用 三星号 或 三下划线 包裹文字，同时加粗和倾斜
plain
**这是加粗文字**，但并非标题。
__这也是加粗文字__，保持段落格式。

*这是斜体文字*，适合轻度强调。
_这也是斜体文字_，保持段落连贯性。

***这是重点强调内容***，适合高亮关键信息。

渲染结果:
这是加粗文字，但并非标题。 这也是加粗文字，保持段落格式。
这是斜体文字，适合轻度强调。
这也是斜体文字，保持段落连贯性。
这是重点强调内容，适合高亮关键信息。

5. 内联 HTML 标签（进阶） ​

plain

<em>这是语义化斜体</em>，常用于专业术语或引用。
