Node ​

指定目录 ​

bash
mkdir /usr/local/node
cd /usr/local/node

安装wget（已安装忽略） ​

bash
sudo yum install wget -y

下载并解压Node ​

访问Node官网（ https://nodejs.org/dist/）下载最新稳定版的Nginx源码包。以下以node16.15.1版本为例 (wget 时多重试几次，有时候会断）

plain
wget https://nodejs.org/dist/v16.15.1/node-v16.15.1-linux-x64.tar.gz
tar -xzf node-v16.15.1-linux-x64.tar.gz --strip-components 1

设置环境变量 ​

bash
echo 'export PATH=/usr/local/node/bin:$PATH' >> ~/.bashrc  
source ~/.bashrc

一键装机 ​

bash
directory_path="/usr/local/node"
rm -rf "$directory_path"
mkdir "$directory_path"
cd "$directory_path"
wget https://nodejs.org/dist/v16.15.1/node-v16.15.1-linux-x64.tar.gz
# wget https://gitee.com/caselibrary/static-resources/raw/main/node-v16.15.1-linux-x64.tar.gz
mkdir nodejs
#  --strip-components 1 解析第一层目录
tar -xzf node-v16.15.1-linux-x64.tar.gz -C nodejs  --strip-components 1

# 设置环境变量
grep -qF 'export PATH="/usr/local/node/nodejs/bin:$PATH"' ~/.bashrc || {  
    echo 'export PATH="/usr/local/node/nodejs/bin:$PATH"' >> ~/.bashrc  
    echo "PATH to Node.js has been appended to ~/.bashrc"  
}  
# 为了让更改在当前终端会话中生效，可以重新加载 ~/.bashrc  
source ~/.bashrc

node 在 windows 系统如何应用 ​

官网下载 node 的 zip 压缩包 https://nodejs.org/dist/v16.15.1/node-v16.15.1-win-x64.zip 或者通过msi镜像文件下载

解压后将当前文件目录在环境变量中指定，便可全局访问。
